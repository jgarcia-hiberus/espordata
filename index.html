Herramienta de validación local

validate.py — CLI pySHACL

Script de línea de comandos para ejecutar las validaciones localmente. Requiere Python 3.8+ y pyshacl.

# Instalar dependencia
pip install pyshacl --extra-index-url https://pypi.org/simple/

# Validar descriptor DCAT
python validate.py --mode dcat --data descriptor-dcat-lote.ttl

# Validar payload de lotes
python validate.py --mode payload --data mi-payload.ttl

# Modo silencioso para CI/CD (exit code 0 = conforme)
python validate.py --mode payload --data mi-payload.ttl --quiet
